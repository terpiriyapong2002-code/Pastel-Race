# Pastel Race

A vibrant and dynamic race simulation game with a beautiful pastel aesthetic.

## Features

- **Dynamic Gameplay**: Fast-paced race simulation.
- **Pastel Aesthetics**: Modern and premium UI design.
- **Progressive Web App (PWA)**: Installable on your device with offline capabilities.
- **Cast Management**: Customize and manage your racers.

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (version 16 or higher recommended)
- [npm](https://www.npmjs.com/)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/[your-username]/pastel-race.git
   ```

2. Navigate to the project directory:
   ```bash
   cd pastel-race
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

### Running the App

To start the development server:
```bash
npm run dev
```

### Building for Production

To create a production-ready bundle:
```bash
npm run build
```

The output will be in the `dist/` directory.

### Running Production Preview

To preview the production build locally:
```bash
npm run preview
```
